AI Resume Screening System

The AI Resume Screening System is an intelligent recruitment automation solution designed to streamline the hiring process. Traditionally, recruiters spend significant time manually reviewing resumes. This project solves that problem by using Artificial Intelligence and Natural Language Processing.

Key Functionalities:

• Automatically reads and analyzes resumes
• Extracts important information such as skills, education, and experience
• Compares candidate profiles with job requirements
• Ranks candidates based on relevance score
• Helps recruiters shortlist the best candidates quickly

Technologies Used:
Python Machine Learning Natural Language Processing Scikit-learn

Impact:
This system significantly reduces manual effort and improves hiring efficiency.
